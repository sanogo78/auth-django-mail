# auth-django-mail
système d'authentification avec django

# Comment utliser ?
D'apres faire:
git clone https://github.com/sanogo78/auth-django-mail/
pour cloner le projet

2. crée une .venv
   Après création du .venv, fais
   : pip install -r requirements.txt | Pour installer toutes les bibliothèques nécessaire à l'exécution du projet

3. Paramètré le fichier .env.exemple dans un fichier .env, pour repondre aux exigence

   Pour plus d'informations sur ce système d'authentification visité la chaîne youtube suivante
   https://youtu.be/u5yIQe8o2zU?si=SQHYX3jL-v0vff3w
